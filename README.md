# Algoritmos Genéticos para Problemas de Optimización

## Descripción
Este proyecto implementa Algoritmos Genéticos utilizando la librería DEAP para resolver tres problemas de optimización:

1. **Optimización de Rutas de Entrega** - Minimizar distancia recorrida por una flota de camiones
2. **Optimización de Circuitos Electrónicos** - Minimizar área y longitud de conexiones en PCB
3. **Optimización de Cartera de Inversiones** - Maximizar rendimiento y minimizar riesgo
